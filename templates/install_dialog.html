Let spack decide which version of bzip2 to install, either from the build cache or from source. Install a package by hash directly. This will either reuse an already installed spec or install from the build cache.
1. Add the mirror {{ copy_command_button() }} 
$ spack mirror add spack-cache https://binaries.spack.io/...
2. Trust the signing key {{ copy_command_button() }} 
$ spack buildcache keys --install --trust --force
