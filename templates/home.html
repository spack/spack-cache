Get started

Run steps 1 & 2 once per Spack instance. Pinned to ....

1. Add the mirror {{ copy_command_button() }} 
$ spack mirror add spack-cache https://binaries.spack.io/...
2. Trust the signing key {{ copy_command_button() }} 
$ spack buildcache keys --install --trust --force
3a. Install a package by name {{ copy_command_button() }} 
$ spack install py-torch
Let spack decide which version of the package to install, either from the build cache or from source.
3b. Or install an exact build by hash {{ copy_command_button() }} 
$ spack install /abc123def456

Install a package by hash directly. This will either reuse an already installed spec or install from the build cache.

  1. 1. Click on the name of the desired package in the sidebar to view a table of builds for that package.
  2. 2. Once you find the desired build, click on the hash to copy it or click the install button for the full install command.
